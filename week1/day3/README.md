# Day 3 - Tokens

## What I Learned

Today I learned about **tokens**.

* A token is a small piece of text that AI reads.
* A token can be a word, part of a word, a number, or a symbol.
* AI counts text in tokens, not words.
* More tokens mean more processing and higher API usage.
* Both the prompt and the AI response use tokens.

### Example

Text:

```text
Hello, ChatGPT!
```

This is about **4–5 tokens** (depending on the tokenizer).

### Summary

Tokens are the basic units that AI models use to understand and generate text.
